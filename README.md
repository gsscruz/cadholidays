# CAN holidays

> An interactive map that displays which provinces of Canada celebrate which holidays\*.

- \*This is not an exhaustive list
- _Holiday data obtained from: https://canada-holidays.ca/api/_

## Techs used

- React
- CSS Modules
- GSAP
- React Simple Maps
